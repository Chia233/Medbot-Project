How it works

How does Medbot AI recommend doctors?

Data Gathering

Our data set consists of thousands of surveys filled up by patients who suffer from common symptoms that has been listed down, along with the doctor's diagnosis.

Database Evaluation

Our database collects the survey and turns them into a machine-readable dataset ready for processing. Once our AI has been trained on predicting the kind of disease that the patient has, the AI is now ready to recommend the appropriate doctor.

Python Processing

Once your survey data has been sent, the Python script executes a test against the data set and sends back an answer that most closely resembles the kind of doctor that you should see

Generating The Result

The data is then displayed to the user when the testing completes
